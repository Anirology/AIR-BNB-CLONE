Airbnb Clone (UI/UX Focus)A frontend responsive web page that clones the user interface and user experience of Airbnb.
This project was developed as part of the Uki program, focusing heavily on modern UI layout, fluid UX interactions, and mobile-first responsiveness using HTML, CSS, and Bootstrap
🚀 Key FeaturesResponsive Grid: Adapts from mobile screens to large desktop monitors.Navigation Bar: Includes a functional search bar capsule and user profile dropdown mockup.
Category Filter: Smooth, horizontally scrollable filter bar with custom icons.Property Cards: Clean card layouts featuring image carousels, ratings, and price formatting
Sticky Footer: Mobile-optimized bottom navigation bar for quick access.UX Enhancements: Micro-interactions, hover effects, and active state transitions.
🛠️ Tech Stack
HTML5: Semantic structure for accessibility.
CSS3: Custom styling, variables, scrollbar hiding, and layout tweaks.
Bootstrap 5: Utility classes, responsive grid system, and component baselines.
FontAwesome / Bootstrap Icons: Scalable vector icons for UI elements.

Use code with caution.🎨 UI/UX Focus Areas1. Typography & HierarchyUsing clean, modern sans-serif fonts to mirror the premium feel of the original platform.
Font sizes and weights are strictly managed to separate property titles from pricing details.
2. Micro-InteractionsHover States: Subtle scaling on property cards.
Active Elements: Visual feedback when selecting categories
Transitions: Smooth color changes on button toggles.
3. Mobile ResponsivenessThe layout prioritizes the mobile view, transitioning into a multi-column grid as viewport size increases.
The desktop top-nav switches seamlessly to a thumb-friendly bottom-nav on smaller screens.
